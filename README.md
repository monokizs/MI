# 🍎 FruitSense AI

Ez a projekt egy mesterséges intelligencia alapú gyümölcsfelismerő rendszer, amely a felismerés után azonnali tápanyagadatokat szolgáltat.

## 🚀 Funkciók
- **Képfelismerés:** TensorFlow/Keras alapú modell (MobileNet).
- **Adatbázis:** JSON alapú tápanyag-adatkezelés.
- **Környezet:** Felhő alapú futtatás Google Colab-ban.

## 🛠️ Felépítés
1. **Adatgyűjtés:** `icrawler` segítségével Bing képeken tanítva.
2. **Tanítás:** Google Teachable Machine.
3. **Implementáció:** Python kód Google Drive integrációval.
